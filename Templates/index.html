NEWS CLASSIFIER

There are 4 classifications: SPORTS, CRIME, SCIENCE AND BUSINESS






ABOUT

News Classifier is a web app that classifies news into 4 categories: Business, Entertainment, Politics, Sports and Technology.
It uses a machine learning model to classify the news. The model was trained on a dataset of 40,000 news articles, the model was trained using a Naive Bayes Classifier.
The model was trained using the scikit-learn library in python. The model was deployed using Flask and Heroku.


Please Note: The model is very weak, it's just for showcase purposes. The results may be innacurate.
